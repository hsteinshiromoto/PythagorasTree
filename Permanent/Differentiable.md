# Differentiable

## Definition

**Definition**. Let $U\subset\mathbb{R}^n$ be an [[open set]] the function 
$$\begin{array}{rcl}
f:&U&\to&\mathbb{R}\\
&x&\mapsto&f(x)\;
\end{array}$$is said to be _differentiable at $a\in U$_ if the limit
$$\displaystyle\lim_{x\to a}\dfrac{f(x)-f(a)-\sum_{i=1}^n b_i(x^i-a^i)}{||x-a||}=0$$
holds.
