# Neighborhood

## Definition

**Definition**. Let $(X, \tau)$ be a [[topological space]], a set $N$ is said to be a _neighborhood_ of a point $p\in X$, denoted as $N(x)$ if there exists an [[open set]] $U$ of $X$ such that $p\in U\subseteq N\subseteq X$.
